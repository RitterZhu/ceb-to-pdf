# 测试 CEB 样本

把至少 1 个 `.ceb` 文件放在本目录下，跑测试时会用到。

## 来源

⚠️ `.ceb` 文件版权属于出版社或作者，**不要**把受版权保护的 CEB 文件 commit 到 GitHub。

### 建议

1. **自己创作的 CEB**（如有）—— 可以放进来
2. **公开版权 / 公有领域 CEB**—— 找 Gutenberg 类似的开源 CEB 库
3. **小测试样本**（10-100 KB）—— Apabi Reader 自带 sample 目录
4. **空跑测试**—— 故意放 0 字节或损坏的 CEB 文件，验证错误处理

## 如果没有 CEB 样本

- 测试会被自动跳过（`pytest.mark.skipif`）
- 仍然可以跑 `python verify.py` 检查其他依赖

## 最小测试样本

至少放 1 个真实的、未加密的 `.ceb` 文件，文件名建议不含特殊字符。
